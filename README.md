# Simple Expense Tracker 💰

A lightweight Command-Line Interface (CLI) application built with Python to help users track their daily expenses on the go. 

---

## ✨ Features
* **Real-time Tracking:** Keeps a running total of your expenses as you enter them.
* **Input Validation:** Built-in error handling (`try-except`) to ensure only valid numbers are accepted, preventing crashes from accidental text inputs.
* **Easy Exit:** Type `quit` at any time to finish tracking and view your final total spent.

## 🚀 How to Run the Project

### Prerequisites
You only need **Python 3.x** installed on your machine.

### Steps:
1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/python-expense-tracker-cli.git
   cd python-expense-tracker-cli
   ```

2. **Run the app:**
   ```bash
   python expense_tracker.py
   ```
